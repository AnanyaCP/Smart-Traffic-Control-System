# Smart Traffic Control System 🚦

![Java](https://img.shields.io/badge/Language-Java-blue)

## 📌 Description
A Java-based multithreaded application that simulates a smart traffic control system. It prioritizes emergency vehicles using a serve-and-resume mechanism and logs events using MySQL (JDBC) and file handling.

---

## 🚀 Features
- Multithreaded traffic simulation  
- Emergency vehicle priority handling  
- Serve & Resume mechanism  
- Real-time vehicle animation (Java Swing)  
- Event logging (Database + File)  

---

## ⚙️ How It Works
- UI starts the system and threads  
- TrafficEngine controls signal logic  
- Animation thread updates vehicle movement  
- Events are logged in database and file  

---

## 🛠️ Technologies Used
- Java (Core + Multithreading)  
- Java Swing (GUI)  
- MySQL (JDBC)  
- File Handling
